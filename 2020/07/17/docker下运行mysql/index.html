docker下运行mysql

1. docker上安装MySQL

2. docker 上启动MySQL，MySQL服务一直运行

（参数 -p 设置端口，–name 取名 ，-e MYSQL_ROOT_PASSWORD=root设置 账号为 root ，密码为 123456 ，

-d 表示作为一个守护进程在后台运行 ）

3. 相关文件地址配置，可以在运行容器时设置：

但要求对应路径对应文件已存在，才能成功挂载相内容到对应位置。

-p 3306:3306：将容器的3306端口映射到主机的3306端口

$HOME/my这个文件夹是存放my.cnf配置文件

-e MYSQL_ROOT_PASSWORD=root：初始化root用户的密码
